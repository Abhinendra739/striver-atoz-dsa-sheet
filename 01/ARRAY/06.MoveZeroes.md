# Move Zeroes

## Problem

Given an integer array `nums`, move all `0`s to the end of the array while maintaining the relative order of the non-zero elements.

The operation must be performed in-place without making a copy of the array.

## Approach

* Use two pointers `i` and `j`.
* `i` points to the position where the next non-zero element should be placed.
* `j` traverses the array.
* Whenever a non-zero element is found, swap it with the element at index `i`.
* Increment `i` after every swap.
* This keeps all non-zero elements at the front while pushing zeroes to the end.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Code

```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int i = 0;

        for(int j = 0; j < nums.size(); j++) {
            if(nums[j] != 0) {
                swap(nums[j], nums[i]);
                i++;
            }
        }
    }
};
```

## Example

Input:

```text
nums = [0,1,0,3,12]
```

Output:

```text
[1,3,12,0,0]
```

Explanation:

* Move all non-zero elements to the front.
* Maintain their relative order.
* Zeroes automatically shift to the end.
