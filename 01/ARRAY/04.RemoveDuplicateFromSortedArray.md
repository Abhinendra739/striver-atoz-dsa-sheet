# Remove Duplicates from Sorted Array

## Problem

Given a sorted array `nums`, remove the duplicates in-place such that each unique element appears only once. Return the number of unique elements.

## Approach

* Use two pointers `i` and `j`.
* `i` keeps track of the last unique element.
* `j` traverses the array.
* Whenever `nums[j]` is different from `nums[i]`, increment `i` and place `nums[j]` at position `i`.
* After the traversal, the first `i + 1` elements contain all unique elements.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Code

```cpp
class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        int i = 0;

        for(int j = 1; j < nums.size(); j++) {
            if(nums[i] != nums[j]) {
                i++;
                nums[i] = nums[j];
            }
        }

        return i + 1;
    }
};
```
