# Missing Number

## Problem

Given an array `nums` containing `n` distinct numbers in the range `[0, n]`, return the only number in the range that is missing from the array.

## Approach

* Use the XOR property:

  * `a ^ a = 0`
  * `a ^ 0 = a`
* XOR all numbers from `1` to `n`.
* XOR all elements of the array.
* XOR the two results together.
* Common elements cancel out, leaving only the missing number.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Code

```cpp
class Solution {
public:
    int missingNumber(vector<int>& nums) {
        int xor1 = 0;
        int xor2 = 0;
        int n = nums.size() + 1;

        for(int i = 1; i < n; i++) {
            xor1 = xor1 ^ i;
        }

        for(int i = 0; i < nums.size(); i++) {
            xor2 = xor2 ^ nums[i];
        }

        return xor1 ^ xor2;
    }
};
```

## Example

Input:

```text
nums = [3,0,1]
```

Output:

```text
2
```

Explanation:

* Numbers should be: `0,1,2,3`
* Missing number is `2`.
